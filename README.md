# FRF-POST-PROCESSING-TOOL
Python based automation tool for correlation of simulation and physical test Frequency Response Function (FRF) data. The tool automatically imports FRF datasets, identifies resonance frequencies, compares simulation and test results, generates correlation reports, and exports graphical results.

Features:-
1. Import simulation and test FRF data from CSV files.
2. Automatic resonance peak detection.
3. FRF comparison plotting.
4. Peak frequency annotation for Simulation to test resonance matching.
5. Identification of unmatched resonances.
6. Frequency error calculation.
7. Automatic correlation report generation.
8. Automatic graph export in .PNG format.

Software Stack
• Python
• Pandas
• NumPy
• Matplotlib
• SciPy

Future Enhancements:-
1. PDF report generation
2. Automatic mode-shape correlation (MAC and FRAC) 
3. Batch processing of multiple FRF datasets
4. GUI-based workflow
5. FEA Software Integration (Ansys, Hypermesh)
